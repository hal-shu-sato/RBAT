# ラジオ配信補助ツール  
## Radio Broadcast Assist Tool  
言語(Language) C# & .NET Core  
開発環境 Visual Studio 2019 & Git & GitHub  

## 機能  
### マイク・PC音・BGMミックス出力
マイク・PC音・BGMの3つの音をソフト上でミキシングできます。<br>
ミキシング後は、仮想サウンドカードより出力します。
### 時報
毎時0分に時報を鳴らすことができます。音源の変更も可能です。
### SEポン出し機能
ショートカットキーによるSEのポン出しが可能です。
### タイマー
指定した時刻に指定した音声ファイルを流すことが可能です。
